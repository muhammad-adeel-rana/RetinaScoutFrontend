Advanced Features for Retinal Diagnostics

A comprehensive web-based application that combines deep learning accuracy with a digitized clinical workflow to reduce observer variability

Pathology Segmentation

Our U-Net model is trained to identify and segment critical features like hemorrhages, exudates, microaneurysms, and the optic disc at the pixel level.

Real-Time Processing

High-resolution patch-based processing ensures detailed segmentation with inference times of just 5-15 seconds on standard GPU hardware.

Clinician Interface

Designed for clarity, the interface provides ophthalmologists with side-by-side visualizations and color-coded overlays to assist in diagnosis.

Role-Based Workflow

The system supports distinct secure roles for Lab Technicians, Doctors, and Patients, facilitating seamless registration and history tracking.

Validated Accuracy

Rigorously trained on the IDRID dataset and evaluated using Dice Coefficient and Intersection over Union (IoU) metrics for robust performance.
